CartFiller SelectorBuilder

Try right here

  • Use this link: SelectorBuilder
  • Your browser will probably complain about popup windows, enable popups and click link again
  • When the popup window will appear look for Search button in the popup window and click it
  • In your main browser window (not popup) point your mouse to the element that you want to build selector for and click
  • You will see DOM tree buttons in the popup window, click them to build selector. The selector is mixture of standard CSS selectors and CartFiller specific methods when CSS selectors are not enough. Would you like specific selectors to be built - let me know.
  • Sometimes target webpage will hide elements when page loses focus. In this case use shift+mousemove instead of click. This works both for Search button and for selecting an element on target page.

On other websites

  • Drag-and-drop the SelectorBuilder link from above to your bookmarks bar.
  • Open target page
