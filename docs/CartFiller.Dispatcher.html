onMainFrameLoaded(watchdog, bubbling)

Handles "main frame loaded" event. If both main frame and worker (job progress) frames are loaded then bootstraps job progress frame If worker have registered onLoad callback, then calls it
Parameters:
Name Type Description
watchdog boolean It may happen, that main frame window code will replace our onLoad handler with its own one and we will miss onLoad event. For such cases UI pings main frame to check whether it is already loaded, and if yes calls this function with watchdog = true. NOTE: as a result this function can be called several times (once) after each ping.
bubbling boolean Means that we received this message from another dispatcher and we should not bubble it
